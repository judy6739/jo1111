أساسيات البرمجة بلغة بايثون

تُعتبر بايثون واحدة من أكثر لغات البرمجة شعبية وسهولة في التعلم، مما يجعلها الخيار الأمثل للمبتدئين. في هذا المقال، سنستعرض أساسيات البرمجة بلغة بايثون ونقدم بعض المفاهيم الرئيسية التي يجب على كل مبتدئ معرفتها.

1. ما هي بايثون؟

بايثون هي لغة برمجة عالية المستوى، تُستخدم في مجالات متعددة مثل تطوير الويب، تحليل البيانات، الذكاء الاصطناعي، وتعلم الآلة. تمتاز بايثون بكونها سهلة القراءة والكتابة، مما يجعلها مثالية للمبتدئين.

2. تثبيت بايثون

قبل البدء في البرمجة باستخدام بايثون، يجب تثبيتها على جهازك. يمكن تحميلها من الموقع الرسمي Python.org. بعد التثبيت، يمكنك استخدام محرر نصوص مثل Visual Studio Code أو PyCharm لكتابة الكود.

3. أول برنامج لك في بايثون

لنبدأ بكتابة أول برنامج بسيط في بايثون الذي يطبع "مرحبا بالعالم":

print("مرحبا بالعالم")

عند تشغيل هذا الكود، سيظهر لك النص على الشاشة. هذا هو أبسط مثال على كيفية استخدام بايثون لطباعة النصوص.

4. المتغيرات وأنواع البيانات

تستخدم المتغيرات لتخزين القيم. في بايثون، يمكنك إنشاء متغيرات بسهولة:

age = 25            # عدد صحيح
name = "علي"       # نص
height = 1.75      # عدد عشري
is_student = True   # قيمة منطقية

5. العمليات الرياضية

يمكنك استخدام بايثون لإجراء عمليات حسابية بسيطة:

a = 10
b = 5
sum = a + b                # جمع
difference = a - b         # طرح
product = a * b            # ضرب
quotient = a / b           # قسمة

print("المجموع:", sum)

6. الشروط

تستخدم الشروط لتنفيذ كود معين بناءً على شروط محددة:

if age >= 18:
    print("أنت بالغ.")
else:
    print("أنت قاصر.")

7. الحلقات

تساعد الحلقات على تكرار تنفيذ كود معين. في بايثون، يمكنك استخدام for وwhile:

# حلقة for
for i in range(5):
    print("الرقم:", i)

# حلقة while
count = 0
while count < 5:
    print("العداد:", count)
    count += 1

8. الدوال

تُستخدم الدوال لتجميع كود في وحدات قابلة لإعادة الاستخدام:

def greet(name):
    return "مرحبًا، " + name

print(greet("علي"))

9. القوائم

تُعتبر القوائم (Lists) من الهياكل الأساسية في بايثون لتخزين مجموعات من العناصر:

fruits = ["تفاح", "موز", "برتقال"]
print(fruits[0])  # طباعة العنصر الأول

10. التعليقات

تستخدم التعليقات لشرح الكود، مما يجعل من السهل فهمه لاحقًا:

# هذا هو برنامج بسيط لحساب مجموع عددين
def add(a, b):
    return a + b

الخاتمة

تعتبر بايثون لغة برمجة رائعة للمبتدئين، حيث تجمع بين البساطة والمرونة. من خلال تعلم أساسيات بايثون، يمكنك البدء في تطوير مهاراتك البرمجية وبناء مشاريع خاصة بك. تذكر أن الممارسة المستمرة هي المفتاح لإتقان البرمجة. استمتع بتعلم بايثون واستكشاف الإمكانيات التي توفرها!
